# 👋 React Native Profile App

[![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactnative.dev/)
[![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)

A simple, clean React Native application that displays user profile information. This is my first React Native project, built to understand the fundamentals of mobile development with React.

## ✨ Features

- 👤 **User Profile Display:** Shows a user's name, role, and bio.
- 📱 **Cross-Platform:** Works on both iOS and Android devices.
- 🎨 **Clean UI:** Centered layout with professional styling.
- 📐 **Flexbox Layout:** Responsive design using React Native's Flexbox system.
- 🔒 **TypeScript:** Type-safe code for better reliability.

## 🛠️ Tech Stack

- **React Native** - Mobile app framework
- **TypeScript** - Type-safe JavaScript
- **React Native CLI** - Development toolchain
- **Flexbox** - Layout system
